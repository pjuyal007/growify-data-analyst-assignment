
# Growify Data Analyst + AI Assignment
End-to-end marketing analytics pipeline using Python, SQL, Power BI, and AI (LLM over SQL)
## Overview
This project demonstrates an end-to-end data pipeline for marketing analytics.

## Pipeline
Raw CSV → Python (Data Cleaning) → SQL Database → Power BI Dashboard + AI Insight Tool

## Tech Stack
- Python (Pandas)
- SQL (SQLite)
- Power BI
- Streamlit
- OpenAI API

## Components
- Data Cleaning (Python)
- Data Modeling (SQL Star Schema)
- Dashboard (Power BI)
- AI Insight Tool (LLM over SQL)

## Key Features
- Cleaned and validated messy marketing data
- Built a star schema (fact + dimension tables)
- Created interactive Power BI dashboard
- Developed AI tool for natural language queries

## How to Run AI Tool
cd ai_tool  
pip install -r requirements.txt  
streamlit run app.py
